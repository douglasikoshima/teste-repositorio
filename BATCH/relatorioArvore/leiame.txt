Uso de relatorioArvore

Linha de comando
relatorioArvore [path/arquivo de configuracao] [path do OutBox]

Onde:
arquivo de configuracao, possue o conteudo abaixo:
usr=C_CGARCIA  -> Usuario de conexao ao Oracle
pwd=C_CGARCIA  -> senha
pth=.
inst=GBLDEVA   -> banco a ser conectado
log=2          -> nivel de log  == 0 Nao LOGA
                                >  0 Ativa Debug  
PATH=/home/mnunes/BATCH/relatorioArvore/InBox   -> diretorio, onde o programa irah 
                                                   pegar os arquivos de variaveis, vindos
                                                   das workstations.
ttlOut=2       -> Tempo de Vida do(s) Arquivo(s) de Saida, quantidade em dias.

1                   Login

 

Está é a tela inicial, onde visualizamos os campos ‘Usuário’, ‘Senha’ e o botão ‘Acessar’. Para acessar o sistema Catálogo de Produtos, Planos e Serviços digite o usuário e senha em seguida clique sobre o botão ‘Acessar’.

 

 

 

 

2       Página Inicial

 

 

Nela encontramos os botões ‘Página Inicial’, ‘Parametrização’, ‘Consultas’, ‘Alterar Senha’ e ‘Sair’.  A qualquer momento podemos voltar a esta tela clicando sobre o botão ‘PÁGINA INICIAL’ ou ‘SAIR’ para sairmos do sistema Catalogo.

 

 

 

3       Parametrização

 

 

3.1  Produtos

 

Nesta etapa iremos  iniciar a Parametrização de Produtos. Inicialmente clique sobre guia ‘PARAMETRIZAÇÃO’ e em seguida sobre ‘PRODUTOS’ ou a qualquer momento ‘SAIR’ para deixar o sistema Catálogo.

                 

 

 

            Ao clicar sobre o ‘PRODUTOS’ a tela apresentará uma série de opções sob este.

 

3.1.1        Parametrização de Características

 

Está funcionalidade permite pesquisar, cadastrar, alterar e excluir características.

 

3.1.1.1  Pesquisar Característica

 

Para pesquisar uma característica digite no campo indicado e depois clique sobre ‘Pesquisar’, o resultado será apresentado na tela em ordem alfabética. Para alterar ou excluir basta clicar sobre o ícone referente ao respectivo procedimento.

 

 

 

3.1.1.2            Incluir Característica

 

Para incluir uma nova característica, clique sobre o botão ‘NOVO’, surgirá na tela um campo para digitação do ‘Nome da Característica’ e a sua ‘Descrição’. Depois de digitado clique sobre o botão ‘Gravar’ para salvar a nova característica ou a qualquer momento sobre o botão ‘Cancelar’ para abortar a operação.

                       

 

Caso não digite o ‘Nome da Característica’ e clique sobre o botão ‘Gravar’ surgirá uma mensagem informando que este é um campo obrigatório. Clique sobre o botão ‘OK’ para retornar a tela.

 

 

 

 

3.1.1.3             Alterar Característica

                                  

Para alterar clique sobre o link ‘Alterar’ associado a uma determinada característica, será exibido o nome e a descrição, informe o novo nome e opcionalmente a descrição  e clique sobre o botão ‘Gravar’ ou a qualquer momento no botão ‘Cancelar’ para abortar a alteração.

                                  

 

3.1.1.4             Excluir Característica

 

Para excluir clique sobre o link ‘Excluir’ associado a uma determinada característica, será exibido será exibido um popup com a lista de todos os modelos associados a essa característica juntamente com a quantidade de produtos de cada modelo, informe a justificativa da exclusão e confirme clicando sobre o botão ‘Sim’ ou ‘Não’ caso queira abortar a exclusão.

 

 

 

3.1.1.5  Valores de uma  Caraterística

 

Esta funcionalidade permite incluir, alterar ou excluir o valor de uma característica.

 

 

3.1.1.5.1        Incluir Valor para a Característica

 

Ao Clicar sobre o link ‘Valores’ associado a uma determinada característica será exibido um popup de valores de características,  informe o novo valor que deseja cadastrar e clique sobre o botão “Incluir” ou a qualquer momento no botão ‘Fechar’ para fechar o popup de valores de características.

 

 

 

3.1.1.5.2          Alterar Valor de uma Característica

 

Ao Clicar sobre o link ‘Valores’ associado a uma determinada característica será exibido um popup de valores de características,  clique sobre o link ‘Alterar’, digite o novo valor para a característica e clique sobre o botão ‘Alterar’. Será exibido um popup para confirmação da alteração  com a lista de todos os modelos associados a essa característica juntamente com a quantidade de produtos de cada modelo. Justifique a alteração e clique sobre o botão ‘Sim’ ou ‘Não’ para abortar a alteração ou  a qualquer momento clique no botão ‘Fechar’ para fechar o popup de valores de características.

 

 

 

3.1.1.5.3                                                                                          Excluir valores de uma característica

 

Ao Clicar sobre o link ‘Valores’ associado a uma determinada característica será exibido um popup de valores de características,  clique sobre o link ‘Excluir’.

 

 

 

           

Será exibido um popup para confirmação da exclusão, clique sobre o botão ‘Sim’ para confirmar ou ‘Não’ para abortar a exclusão ou  a qualquer momento clique no botão ‘Fechar’ para fechar o popup de valores de características.

 

 

 

3.1.2        Parametrização de Modelos

 

Está funcionalidade permite consultar, cadastrar, alterar e excluir modelos.

 

3.1.2.1  Consultar Modelos

 

Para a consulta de modelos, deverá ser obrigatoriamente informado o filtro tipo de produto e, opcionalmente, o fabricante, ou tecnologia, ou característica e valores ou ainda o nome do modelo (no mínimo as 2 primeiras letras). Selecionados os filtros, clicar sobre o botão “Pesquisar”.

 

 

 

 

Caso não sejam informados os campos obrigatórios será exibido um popup alertando, clique no botão ‘OK’ para retornar a tela e informá-los.

 

 

 

 

 

 

 

 

 

 

Será exibida então uma lista com os nomes dos modelos localizados bem como seus respectivos fabricantes e também os links Características, Imagem, Alterar, Copiar, excluir e detalhes do modelo.

 

 

 

 

3.1.2.2  Características de um Modelo

 

Ao clicar sobre o link ‘Características’ associado a um determinado modelo será exibido um popup de características de um Modelo.

 

 

 

3.1.2.3                                                                       Imagem de um Modelo

 

Ao clicar sobre o link ‘Imagem’ associado a um determinado modelo será exibido um popup de imagem de um Modelo.

Para adicionar uma nova imagem, salve a imagem em um diretório local, selecione o arquivo através do botão ‘Procurar’ e clique sobre o botão ‘Adicionar’.

A imagem deve estar no formato JPEG ou GIF com dimensões 110 px de largura x 165 px de altura.

 

 

Para visualizar uma imagem basta clicar sobre o nome da mesma.

 

 

3.1.2.4             Incluir Modelo

 

Para incluir um novo Modelo, clique sobre o botão ‘NOVO’na tela principal de ‘Parametrização de Modelos’, surgirá então um campo para seleção do ‘Tipo de Produto’, ‘Fabricante’, ‘Tecnologia/Freqüência’ e outro campo para digitação do ‘Nome comercial’. Depois de devidamente preenchidos os campos clique sobre o botão ‘Gravar’ para salvar o novo Modelo ou ‘Cancelar’ para abortar a operação.

 

 

           

 

                                  

 

Selecione as devidas tecnologias e freqüências e depois clique sobre o botão ‘OK’.

 

 

 

 

3.1.2.5             Excluir Modelo

 

Ao clicar no link “Excluir” associado a um determinado modelo, será exibido um popup com a lista de todos os códigos SAP de produtos associados a esse modelo.

 

 

É obrigatório informar a justificativa para exclusão antes de confirmá-la. Caso não tenha sido informado, será exibida uma mensagem informativa.

 

 

 

 

3.1.2.6.1                    Alterar Modelos

 

Ao clicar no link “Alterar” associado a um determinado modelo, será exibido na tela  um campo para seleção do ‘Tipo de Produto’, ‘Fabricante’, ‘Tecnologia/Freqüência’ e outro campo para digitação do ‘Nome comercial’. Depois de devidamente preenchidos os campos clique sobre o botão ‘Gravar’ para salvar a alteração do Modelo ou ‘Cancelar’ para abortar a operação.

 

 

 

 

3.1.2.7    Consultar Detalhes

 

Ao clicar no último link exibido no grid de modelos, serão exibidos os detalhes de determinado modelo. Para sair clique no botão ‘Fechar’.

 

 

 

 

 

3.1.2.8             Copiar Modelo

 

Essa funcionalidade permite que se crie um novo modelo a partir de um já cadastrado e é útil quando se deseja cadastrar um modelo bastante similar a outro que já existe. Ao clicar no link “Copiar” associado a um determinado modelo, essa funcionalidade será ativada.

Os campos exibidos serão os mesmos da funcionalidade “Incluir Modelo”, descrita anteriormente, com a diferença de que virão preenchidos, com exceção, do nome do modelo que virá em branco para que seja preenchido com o novo nome. O campo tipo de produto não estará habilitado para edição, visto que deverá ser mantido o mesmo do modelo que foi usado como espelho. Essa funcionalidade permitirá que sejam duplicados as características e valores do modelo original, porém, não serão duplicadas as associações com imagens, produtos e variáveis de classificação.

Informados os dados, o usuário deverá clicar no botão ‘Gravar’ ou ‘Cancelar’ caso queira abortar a cópia de Modelo.

 

 

 

 

3.1.2.9             Alterar Indicador de Disponibilidade para Sistema Externo

 

Ao clicar no checbox “Disp” do grid  poderá ser alterado o indicador de disponibilidade para o sistema externo para determinado modelo.

 

 

 

 

3.1.2.10           Alteração de Indicador de Fim de Vida do Modelo:

 

Ao clicar no checkbox “Fim de Vida” do grid poderá ser alterado o indicador de que o modelo não mais será produzido pelo fornecedor.

 

 

 

 

3.1.3        Parametrização de Produtos

 

Está funcionalidade permite consultar, associar e desassociar produtos com modelos.

 

 

3.1.3.1                         Pesquisar Produtos

 

Para pesquisa de produtos, deverão ser obrigatoriamente informados os filtros tipo de produto e código associado (quando selecionada a opção “Não”, o filtro “Modelo” deve ser desabilitado). O filtro fabricante será obrigatório para todos os tipos de produto, com exceção, dos tipos “acessório” (código do tipo de produto no banco de dados igual a 10), “cartão de recarga” (código do tipo de produto no banco de dados igual a 9) e “sim card” (código do tipo de produto no banco de dados igual a 7). O filtro tecnologia será obrigatório para todos os tipos de produto, com exceção, dos tipos  “acessório” (código do tipo de produto no banco de dados igual a 10), “cartão de recarga” (código do tipo de produto no banco de dados igual a 9) e “sim card” (código do tipo de produto no banco de dados igual a 7) para os quais o campo será ocultado. Clique sobre o botão ‘Pesquisar’.

 

 

 

 

 

            Caso não seja informado os campos obrigatórios o sistema exibe uma mensagem informativa. Clique no botão ‘OK’ para retornar a tela.

 

 

 

 

 

 

Preencher os campos obrigatórios e clicar sobre o botão ‘Pesquisar’. Serão dois tipos diferentes de pesquisa, um com outro sem   ‘Código associado’.

             

 

 

 

 

 

 

3.1.3.1.1                   Pesquisar Produtos  sem ‘código associado’

 

Esta pesquisa exibirá somente os produtos que não  possuem ‘códigos associados’.

 

 

 

3.1.3.1.2                               Pesquisar Produtos  com ‘código associado’

 

Esta pesquisa exibirá somente os produtos que já possuem ‘códigos associados’.

 

 

 

 

                       

3.1.3.2                         Alterar  Indicador de Disponibilidade para Sistema Externo

 

                       

Ao clicar no checbox “Disp” do grid,  poderá alterar o indicador de disponibilidade para o sistema externo para determinado produto.

 

 

 

 

3.1.3.3                         Associar  Produto a Modelo

 

Para associação de um ou mais produtos a modelo, primeiramente deverá realizar a pesquisa SEM “Código Associado” (3.1.3.1.1), após essa etapa deverá clicar no checkbox exibido no grid de produtos ao lado do código SAP do produto. Poderão ser selecionados um ou mais produtos. Feito isso deve-se clicar no botão “Associar”.

 

 

 

 

Ao clicar no botão “Associar”, será exibido um grid com os modelos disponíveis para associação com os produtos selecionados. Serão exibidos somente modelos que contenham o mesmo tipo de produto, fabricante e tecnologia dos produtos selecionados. Para os casos em que a informação de fabricante e tecnologia não são obrigatórios (tipos de produto “acessório”, “cartão de recarga” e “sim card“), a mesma não será considerada na busca de modelos disponíveis para associação. Após selecionar o(s) modelo(s) que deseja associar ao(s) produtos(s) previamente selecionados clique no botão “OK”.

 

 

 

 

 

3.1.3.4                         Remover Associação de Produto a Modelo

 

Para remover a associação de um ou mais produtos de seus respectivos modelos, primeiramente deverá realizar a pesquisa COM “Código Associado” (3.1.3.1.2), após essa etapa deverá selecionar um ou mais produtos, no grid de produtos, via checkbox localizado ao lado do código SAP do produto e clicar no botão “Remover Associação”.

 

 

 

 

 

 

Ao clicar no botão “Remover Associação”, será exibido um popup de confirmação e para preenchimento do campo obrigatório “Justificativa”.

Caso preencha a justificativa e clique no botão “SIM” será feita a remoção de associação de produto a modelo. Caso opte em desistir da remoção clique no botão “Não”.

 

 

 

 

3.1.3.5                         Alterar Associação de Produto a Modelo

 

Para alterar a associação de um ou mais produtos de seus respectivos modelos, primeiramente deverá realizar a pesquisa de produtos  COM “Código Associado” (3.1.3.1.2), após essa etapa deverá selecionar um ou mais produtos, no grid de produtos, via checkbox localizado ao lado do código SAP do produto e clicar no botão “Alterar”.

 

 

 

 

Ao clicar no botão “Alterar”, será exibido um grid com os modelos disponíveis para associação com os produtos selecionados. Serão exibidos somente modelos que contenham o mesmo tipo de produto, fabricante e tecnologia dos produtos selecionados. Para os casos em que a informação de fabricante e tecnologia não são obrigatórios (tipos de produto “acessório”, “cartão de recarga” e “sim card”), a mesma não será considerada na busca de modelos disponíveis para associação. Após selecionar o(s) modelo(s) que deseja associar ao(s) produtos(s) previamente selecionados clique no botão “OK”.

 

 

 

 

 

 

 

3.1.4                               Parametrização de Tecnologia

 

Essa funcionalidade permiti consultar, cadastrar e excluir tecnologias.

 

3.1.4.1                         Consultar Tecnologia

 

Ao carregar a tela, deverá ser apresentada a lista de todas as tecnologias bem como tecnologia de referência associada, em ordem alfabética pelo nome da tecnologia.

 

 

 

 

 

3.1.4.2                         Excluir Tecnologia

 

Ao clicar no link “Excluir” associado a uma determinada tecnologia,  será exibido um popup com a lista de todos os modelos associados a essa tecnologia juntamente com a quantidade de produtos de cada modelo.

 

 

 

 

 

 

            O sistema exibirá uma mensagem de confirmação da exclusão. Clique sobre o botão ‘Sim’ para confirmar ou ‘Não’ para abortar a exclusão.

 

 

 

 

                                  

3.1.4.3                         Incluir Tecnologia

 

 

Ao clicar no botão “Novo”, o usuário deverá obrigatoriamente informar o nome da nova tecnologia que deseja cadastrar e a tecnologia de referência associada. A lista de tecnologias de referência cadastradas será informada via combo. Informados esses dados, o usuário deverá clicar no botão “Gravar”.

 

 

 

 

 

                                   3.1.5                Parametrização de Tipo de Freqüências

 

                                                           Essa funcionalidade permiti ao usuário consultar, cadastrar e excluir tipos de frequência.

 

 

3.1.5.1                         Consultar Tipo de Freqüência

 

 

Ao carregar a tela, deverá ser apresentada a lista de todos os tipos de frequências cadastrados.

 

 

 

3.1.5.2                         Excluir Tipo de Freqüência

 

Ao clicar no link “Excluir” associado a um determinado tipo de frequência, será exibido um popup com a lista de todos os modelos associados a esse tipo de frequência juntamente com a quantidade de produtos de cada modelo. Informe a justificativa e clique sobre o Botão ‘Sim’ ou ‘Não’ para abortar a exclusão.

 

 

 

 

3.1.5.3                         Incluir Tipo de Freqüência

 

 

Ao clicar no botão “Novo”,  deverá ser obrigatoriamente informado o nome do novo tipo de freqüência que deseja cadastrar e a quantidade de valores de frequência associada. Informados esses dados, o usuário deverá clicar no botão “Gravar” para finalizar a inclusão ou ‘Cancelar’ para abortar a inclusão.

 

 

 

 

 

3.1.6                               Parametrização de Freqüências

 

Essa funcionalidade  permiti  consultar, cadastrar e excluir valores de freqüência.

 

3.1.6.1                         Consultar Freqüência

 

 

Ao carregar a tela, deverá ser apresentada a lista de todos os valores de frequência cadastrados.

 

 

 

 

 

3.1.6.2                         Excluir Freqüências

 

Ao clicar no link “Excluir” associado a um determinado valor de frequência, será exibido um popup com a lista de todos os modelos associados a esse valor de frequência juntamente com a quantidade de produtos de cada modelo. Deverá ser informada a justificativa da exclusão antes da confirmação, após a justificativa clique no botão ‘Sim’ ou ‘Não’ para abortar a exclusão da tecnologia.

 

 

 

 

 

3.1.6.3                         Incluir Freqüências

 

Ao clicar no botão “Novo”, deverá ser obrigatoriamente informado o valor da nova frequência que deseja. Informado esse dado, clique sobre o botão “Gravar” ou ‘Cancelar’ para abortar a inclusão de um novo valor de freqüência.

 

 

 

 

 

 

 

 

3.1.7                          Parametrização de Associação de Tecnologia com Tipo de Freqüência

 

Essa funcionalidade permiti consultar, cadastrar, alterar e excluir associação de tecnologia com tipo de freqüência e dessa com valores de freqüência.

 

           

3.1.7.1                         Consultar Associação de Tecnologia com tipo de Freqüência

 

 

Ao carregar a tela, deverá ser apresentada a lista de todas as associações de tecnologia e tipo de frequência parametrizadas no sistema, em ordem alfabética pelo nome da tecnologia.

 

 

 

 

 

 

3.1.7.2                         Incluir Associação de Tecnologia com Tipo de Freqüência

 

Ao clicar no botão “Novo”,  deverá ser obrigatoriamente informado, o nome da tecnologia e o tipo de frequência que deseja associar. Para ambos os dados será exibido um combo com a lista de opções existentes. Informados esses dados, clique sobre o botão “Gravar” ou ‘Cancelar’ para abortar a inclusão de uma nova Associação de Tecnologia com Tipo de Freqüência.

 

 

 

 

3.1.7.3                         Alterar Associação de Tecnologia com Tipo de Freqüência

 

 

Ao clicar no link “Alterar” associado a uma determinada tecnologia e tipo de freqüência, deverá ser  informado obrigatoriamente, o nome da tecnologia e/ou o tipo de frequência como o novo valor. Para ambos os dados, serão exibidos em um combo com a lista de opções existentes, sendo que o valor selecionado “por default” será o da associação selecionada para alteração Informados esses dados, clique sobre o botão “Gravar” ou ‘Cancelar’ para abortar a alteração de uma nova Associação de Tecnologia com Tipo de Freqüência.

 

 

 

 

 

 

3.1.7.4                         Exclusão de Associação de Tecnologia com Tipo de Freqüência

 

 

Ao clicar no link “Excluir” associado a uma determinada associação de tecnologia com tipo de frequência, será exibido um popup com a lista de todos os modelos associados a essa tecnologia e tipo de frequência juntamente com a quantidade de produtos de cada modelo. Deverá ser informado a justificativa para exclusão antes de confirmá-la. Clique no botão ‘Sim’ para confirma ou ‘Não’ para abortar a exclusão da Associação de Tecnologia de Tipo de Freqüência.

 

 

 

 

 

 

 

3.1.7.5                         Alteração/Inclusão/Exclusão de Valor de Frequência para a Associação de Tecnologia com Tipo de Freqüência

 

 

Essa funcionalidade é ativada a partir do popup de valores de frequência, ativado ao clicar no link “Frequência” associado a uma determinada tecnologia e tipo de frequência. O popup também será exibido automaticamente após a gravação de uma alteração de associação de tecnologia com tipo de freqüência. Serão exibidos todos os valores de freqüência existentes, com um checkbox ao lado esquerdo do valor. Os valores já associados estarão checados. Poderão ser selecionado(s) um ou mais valores para associação ou desassociação. Deverá ser selecionados os valores e clicar no botão “OK”.

 

 

                        No caso da desassociação o sistema exibirá um popup com a confirmação da Desassociação da freqüência. Clique sobre o botão ‘Sim’ para confirmar ,  ‘Não’ ou  ‘Fechar’ para abortar a exclusão do valor da freqüência.

 

 

 

 

 

 

3.1.8    Parametrização de Formas e Condições de Pagamento

 

Essa funcionalidade permite consultar e alterar as formas e condições de pagamento, parametrizáveis por canal, plataforma e tipo de produto.  Permite, também, exportar os dados de formas e condições de pagamento para um arquivo CSV.

 

 

3.1.8.1          Pesquisa e alteração de Formas e Condições de Pagamento

 

Para a pesquisa de formas e condições de pagamento, deverão ser obrigatoriamente informados os filtros tipo de produto e canal. código associado (quando selecionada a opção “Não”, o filtro “Modelo” deve ser desabilitado). O filtro plataforma será obrigatório para todos os tipos de produto, com exceção, dos tipos “acessório” (código do tipo de produto no banco de dados igual a 10) e “cartão de recarga” (código do tipo de produto no banco de dados igual a 9).

 

 

 

 

 

 

                        Caso não seja informado  ‘Tipo de Produto’ e/ou ‘Plataforma’ e/ou ‘Canal’, que são campos obrigatórios o sistema exibirá uma mensagem informativa dos campos obrigatórios. Clique sobre o botão ‘OK’ para retornar  e preencher corretamente os campos obrigatórios.

 

 

 

Selecionados os filtros corretamente e  clique sobre o botão “Pesquisar”. Será exibido as formas e condições de pagamento.

            Caso,  queira fazer alguma alteração, deverá selecionar as formas e condições que deseje habilitar e, então, clicar no botão “Salvar”

 

 

 

 

 

3.1.8.2                         Exportar Dados para arquivo CSV

                                         3.1.9.1             Pesquisa e alteração do Número Máximo de Parcelas para Forma de Pagamento Financeira

 

 

Ao selecionar os filtros “Canal” e “Forma de Pagamento” (como Financeira, nesse caso), será exibido o campo: “Número Máximo de Parcelas”, campo numérico, preenchido com valor já parametrizado na base ou em branco caso ainda não tenha sido configurado um valor.

 

 

 

Poderá alterar esse valor e clicar no botão “Salvar”. Será validado se o campo foi preenchido e, caso não tenha sido, será exibida uma mensagem de alerta.

 

 

 

 

 

3.1.8.3                         Parametrização do Desconto para Forma de Pagamento Cartão de Crédito

 

 

Ao selecionar os filtros “Canal” e “Forma de Pagamento” (como Cartão de Crédito, nesse caso), serão exibidos: “Número Máximo de Parcelas”, campo numérico, preenchido com valor já parametrizado na base ou em branco caso ainda não tenha sido configurado um valor “Taxa de Juros Mensal”, campo numérico, preenchido com valor já parametrizado na base ou em branco caso ainda não tenha sido configurado um valor, “Número Máximo de Parcelas Sem Juros”, campo numérico, preenchido com valor já parametrizado na base ou em branco caso ainda não tenha sido configurado um valor, “Tabela de Desconto Progressivo”, com valores já previamente calculados caso já tenha sido parametrizado anteriormente

 

 

 

 

 

 

Ao clicar no botão “Calcular”, a camada de apresentação fará o cálculo do desconto e exibirá o resultado na tela. Caso clique em “Salvar”, os valores do desconto serão gravados na base de dados.

 

 

 

3.1.8.4                         Exportar dados para arquivo CSV
